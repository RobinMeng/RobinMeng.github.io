    网络体系结构

    网络原理
    Created At : 2020-10-09 10:52
    Count:352 Views 👀 :

    2. 协议三要素
    3. ISO在制定OSI/RM 对层次划分遵循的原则
    4. OSI/RM 数据传输的过程
    5. OSI/RM 与TCP/IP 的异同点
    1. 分层 是系统分解的最好方法之一
    2. 通信服务分为两大类 分别是 面向连接服务 和 无连接服务
    3. TCP/IP 协议是基于ARPANET 互联网开发的
    4. 网络协议是为计算机网络中进行数据交换而建立的规则 标准 或 约定集合

    计算机层次化体系结构的特点

    1. 除了物理介质实通信外 其余对等协议之间都是虚通信
    2. 对等层的虚通信必须遵守该层协议
    3. n层的虚拟通信 是通过 n/n-1层接口出 n-1层提供的服务及通信来实现

    协议三要素

    1. 语法
    2. 语义
    3. 定时

    ISO在制定OSI/RM 对层次划分遵循的原则

    1. 每层功能应明确 并且相互独立
    2. 层间接口必须清晰 跨越接口的信息应尽可能少
    3. 层数应适中

    OSI/RM 数据传输的过程

    1. 发送进程发送给接收进程的数据 实际上经过发送方各层从上到下传递到物理介质
    2. 通过物理介质传输到接收方后 在经过从下到上各层传递 最后到达接收进程

    OSI/RM 与TCP/IP 的异同点

    1. 相同
      1. 协议栈为基础
      2. 彼此协议独立
      3. 分层
    2. 不同
      1. OSI 七层 TCP 四层
      2. 无连接和面向连接通信范围不同
